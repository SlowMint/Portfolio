F.A.L.L.

Friends of Apex Legends Leaderboard is a cross platform friendslist that allows users to display player stats, skins, etc. all in one place. This was done by retrieving JSON from a fan made Apex Legends API and implementing a CRUD database for its functionality. This application was built entirely in .NET Core and uses Dapper backend with MySQL.

myVenues

myVenues is an ecommerce site created out of a need of a personal friend. They are an artist who plays at local bars and needed a means of displaying some of their content, personal information, and perfomance schedules. This application is mobile responsive. It includes payment methods, analytics, 3rd party authentication, and authorization. The technologies used in this application where .NET Core, Bootstrap 5, Azure Cloud Services, and Entity Framework.
